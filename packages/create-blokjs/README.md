# create-blokjs

Scaffold a new BlokJS project in seconds.

## Usage

```bash
npm create blokjs my-app
```

Then follow the printed instructions:

```bash
cd my-app
npm install
npx vite
```

## What you get

```
my-app/
  index.html
  main.js
  package.json
  vite.config.js
  components/
    counter.js
  stores/
    app.js
```

A minimal starter with a counter component, an app store, and Vite + vite-plugin-blokjs preconfigured.
